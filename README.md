## Job Notification App — Design System Foundation

This repository contains the **design system foundation** for a premium SaaS-style Job Notification App.

### Principles
- Calm, intentional, coherent, confident
- No gradients, no glassmorphism, no neon, no animation noise

### Constraints implemented
- **Spacing scale only**: 8, 16, 24, 40, 64px
- **Typography**:
  - Headings: system serif
  - Body: system sans, 17px base, \(line-height: 1.7\)
  - Text blocks max width: 720px
- **Color discipline**:
  - Background: `#F7F6F3`
  - Text: `#111111`
  - Accent: `#8B0000`
  - Success: muted green

Note: To honor the “maximum 4 colors across the UI” constraint, **warning styling uses the accent red** rather than introducing a separate amber hue.

### Run locally
Open `index.html` in a browser.
